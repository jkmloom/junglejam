# shui-lang

A fun and functional compiled programming language inspired by Python and C++.

## Overview

shui-lang is designed for learning, experimentation, and exploring core concepts in language and compiler development. It aims to blend Python-like readability with C++-style type declarations, making coding both exciting and accessible.
